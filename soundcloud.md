# SoundCloud tools

## Install work communication tools

```console
yay -S --needed \
    bluejeans \
    slack-desktop \
    zoom
```

## Install VPN

1. Ensure openvpn NetworkManager plugin is installed `yay -S networkmanager-openvpn openvpn`
2. Add new VPN profile to NetworkManager using the `*.ovpn` file provided by company
